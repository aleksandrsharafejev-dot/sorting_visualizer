Sorting Visualizer — Project Overview

Academic submission summary.

Abstract

Sorting Visualizer is an academic demonstration project that explores how classic sorting algorithms behave on arrays of varying sizes. The system provides real-time animation, user-adjustable parameters, and live metrics to highlight algorithmic complexity and operational cost.

Goals

  • Provide an intuitive visual explanation of sorting mechanics.
  • Allow users to tune input size, value range, and animation speed.
  • Compare algorithmic strategies using operational metrics.

Features

  • Interactive web interface with algorithm selection.
  • Live animation of array mutations.
  • Runtime metrics: comparisons, swaps, elapsed time.
  • Export visualization frames as PNG for reports.

Algorithms Implemented

Bubble Sort repeatedly swaps adjacent out-of-order elements.

Quick Sort partitions around a pivot, then recursively sorts.

Merge Sort splits the array and merges sorted halves.

Complexity Summary

  • Bubble Sort: O(n^2) comparisons, O(n^2) swaps in worst case.
  • Quick Sort: O(n log n) average, O(n^2) worst case.
  • Merge Sort: O(n log n) time, O(n) extra memory.

Metrics Explained

  • Comparisons: element-to-element comparisons.
  • Swaps: element exchanges or placements.
  • Elapsed: wall-clock time in seconds.

Technology

HTML/CSS/JavaScript with Canvas rendering.

Author Statement

This project was prepared as an academic demonstration of algorithms.
