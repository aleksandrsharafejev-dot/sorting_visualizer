# Sorting Algorithm Visualizer

This project implements and visualizes several fundamental sorting algorithms in Python.

The visualizations can be opened through the provided index.html file in the project.

## Concepts Covered

* Algorithm design
* Time complexity (Big O)
* Recursion
* Data visualization

## Implemented Algorithms

* Bubble Sort
* Merge Sort
* Quick Sort

## Concepts Covered

* Algorithm design
* Time complexity (Big O)
* Recursion
* Data visualization

## Purpose

The purpose of this project is to understand how sorting algorithms work internally and to visualize their behavior on different datasets.
